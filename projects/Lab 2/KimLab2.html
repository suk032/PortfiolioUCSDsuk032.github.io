How to define a table?
A table is created in HTML by using the opening table and closing table tags.

Describe when to use the following HTML elements:

<tr>
The <tr> element is used as an opening and closing tag when wanting to define a row in a table within the table element.
<th>
The <th> element is used as an opening and closing tag within the <tr> tag to define a cell as a header of the cells. The <th> element also bolds the text in the header cell.
<thead>
The <thead> element is used as an opening and closing tag in order to group header cells of a table together.
<tfoot>
Similarly to the <thead> element, <tfoot> is also used as an opening and closing tag in order to group closing cells of a table together.
<caption>
The <caption> element is used as a tag after the opening table tag in order to create a caption for the table, which is centered.
What is the difference between colspan and rowspan?
colspan is an attribute used to create a cell over multiple columns whereas rowspan is used to create a cell over multiple rows in a table.
What is the difference between cells spacing and padding?
Cells spacing is used to define the space between cells whereas cell padding is used to define the space of a cell's content and the cell's border.
How do you define table borders using CSS?
To define table borders using CSS, the border property must be used after the table element. If borders within the table are wanted, the elements should be named as <table>, <th>, <td> with an opening bracket.
How do you control border widths using CSS?
To control border widths using CSS, add width: __ with a percentage corresponding to how much the table should fill up the screen to the table.
